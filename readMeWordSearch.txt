This program takes in input for a wordsearch puzzle.
The format file explains how the text file needs to be formatted.
The wordsearch.txt file is an example of actual input. To use the program your input needs to look like this.
To use the program the text file needs to be named wordsearch.txt and nothing else.
The wordsearch.txt file needs to be in the same folder as the wordsearch.jar for the program to work.
